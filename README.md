# 🛒 FreshMart - Online Grocery Store

FreshMart is a responsive and user-friendly grocery shopping website built using **HTML, CSS, and JavaScript**. It allows users to browse grocery categories, view special offers, add products to a shopping cart, and manage their purchases through an interactive interface.

---

## 📌 Features

### 🏠 Home Page
- Attractive landing page with video background
- Featured grocery categories
- Product highlights and promotional offers
- Customer reviews section
- Responsive design

### 🔐 Login & Signup
- User authentication interface
- Toggle between Login and Registration forms
- Clean and modern UI

### 🥭 Product Categories
- Fresh Fruits
- Dairy Products
- Vegetables
- Snacks

### 🛒 Shopping Cart
- Add products to cart
- Update product quantities
- Remove products
- Calculate total price
- Store cart data using Local Storage

### 🎁 Special Offers
- Discounted grocery products
- Search functionality
- Promotional banners

### 💬 Contact & Support
- Chat-style customer support interface
- Interactive messaging system
- Simulated owner/customer communication

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Local Storage API
